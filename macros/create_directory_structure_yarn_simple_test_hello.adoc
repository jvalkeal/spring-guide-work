=== Create the directory structure

In a project directory of your choosing, create the following subdirectory structure; for example, with `mkdir -p src/test/java/hello` on *nix systems:

    └── src
        ├── test
        │   └── java
        │       └── hello
        └── main
            ├── resources
            └── java
                └── hello
                    ├── client
                    ├── appmaster
                    └── container
